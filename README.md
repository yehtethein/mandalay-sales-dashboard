# Mandalay Electronic Sales Dashboard

## Overview
This project presents an electronic sales dashboard for the Mandalay Region.

## Key Metrics
- Total Sales: 3 Billion
- Total Profit: 779.46 Million
- Total Units Sold: 30 Million
- Profit Margin: 30.10%

## Features
- Sales trend over time
- Product performance analysis
- Discount impact analysis
- Township-level insights
